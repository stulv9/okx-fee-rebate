## 二、欧易佣金到底是什么：先搞清楚你在付哪些钱

欧易佣金，简单说就是你在欧易平台上交易时付给平台的手续费。它不是一个单一费率，而是按你做什么操作、交易量多少、持仓多少OKB来动态划分的。

**主要分这么几类：**

- **现货交易费**：买卖比特币、以太坊这些主流币，普通用户挂单0.08%、吃单0.10%
- **合约交易费**：永续合约和交割合约，普通用户挂单0.02%、吃单0.05%
- **期权交易费**：BTC/ETH期权，普通用户挂单0.02%、吃单0.03%
- **交割手续费**：到期交割统一收0.01%，盘前交割1%
- **强平手续费**：按你当前等级的吃单费率收
- **提币手续费**：链上提币按币种网络费率收，跟交易费是两回事

这里有个概念很多人搞混：**挂单和吃单**。

挂单就是你设个价格挂到订单簿上等别人来成交，给市场提供流动性，费率低；吃单就是你直接按市价单成交，消耗流动性，费率高。所以同一笔交易，挂单和吃单的成本可能差一倍多。会做的人都会尽量用限价单挂单，能省不少。

欧易的费率不是固定的，会根据你过去30天的交易量和资产量每天动态更新一次（凌晨4:00-6:00 UTC+8）。这就是它的VIP等级体系，下面专门讲。
